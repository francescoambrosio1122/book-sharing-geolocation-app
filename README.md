# Book Sharing Geolocation App

Progetto sviluppato nell’ambito del Project Work del corso di laurea in Informatica per le Aziende Digitali (L-31).

## Descrizione

L’applicazione è progettata per consentire la condivisione del patrimonio librario tra utenti privati attraverso un sistema di geolocalizzazione.  
L’obiettivo è mettere in rete libri inutilizzati, favorendo la diffusione della lettura e la collaborazione tra utenti.

## Architettura del sistema

L’applicazione è progettata secondo un’architettura client-server suddivisa in tre livelli:

- **Frontend**: interfaccia utente accessibile via browser
- **Backend**: gestione della logica applicativa e delle API
- **Database**: archiviazione dei dati relativi a utenti, libri e richieste

## Funzionalità principali

- Registrazione e autenticazione utenti
- Inserimento e gestione dei libri
- Ricerca geolocalizzata dei libri disponibili
- Gestione delle richieste di prestito tra utenti

## Tecnologie utilizzate (ipotesi progettuale)

- Frontend: HTML, CSS, JavaScript
- Backend: Node.js / Express
- Database: MySQL o MongoDB
- API: REST
- Geolocalizzazione: Google Maps API

## Infrastruttura

Il sistema è progettato per essere distribuito su infrastruttura cloud, al fine di garantire scalabilità, disponibilità e sicurezza del servizio.
